# Gravity Sim

### Description

It's a simple gravity simulation and not really a game.

### Features

 - Gravity

### Controls

Keyboard:
- Period and Comma to increase and decrease time scale.
- Space to pause the simulation.

Mouse:
- Click & Drag to move the camera around.
